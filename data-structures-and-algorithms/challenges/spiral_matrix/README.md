# Spiral Matrix

## Challenge

Given a positive integer n, generate an n x n matrix filled with elements from 1 to n2 in spiral order.

## Approach & Efficiency

This method can be described as O(n<sup>2</sup>) time and O(n<sup>2</sup>) space complexity
